# Alyona- Full Stack E-Commerce Platform

## Overview

Alyona is a modern full-stack E-Commerce web application built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The platform provides a seamless shopping experience for customers and a powerful admin dashboard for managing products, categories, orders, and users.

## Features

### Customer Features

* User Registration & Login
* JWT Authentication & Authorization
* Product Browsing
* Product Search & Filtering
* Product Details Page
* Shopping Cart Management
* Wishlist Functionality
* User Profile Management
* Order Placement (Cash on Delivery)
* Order History

### Admin Features

* Secure Admin Dashboard
* Product Management (Create, Update, Delete)
* Category Management
* User Management
* Order Management
* Order Status Updates
* Sales Statistics Dashboard

### Media Management

* Cloudinary Image Upload
* Product Image Management
* User Profile Picture Upload
